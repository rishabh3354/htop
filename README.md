**ABOUT APPLICATION :**

Monitor your computer's CPU and RAM usage in real time with a fast and lightweight desktop application designed for Linux. 
Track processor utilization, memory consumption and overall system performance through beautiful live graphs and an intuitive interface. The application is optimized to use minimal system resources while providing accurate performance statistics.

**Features:** 

* Real-time CPU usage monitoring 
* Live RAM and memory usage statistics 
* Interactive performance graphs 
* Low CPU and memory overhead 
* Lightweight and responsive interface 
* Instant system resource monitoring 
* Internet upload and download speed meter 
* Designed for Linux desktops

**NOTE**

If CPU Temp is not Coming/not accurate in the System Monitor, then run below command in your Terminal/Shell.

    sudo snap connect htop-pro:hardware-observe

## Install System Monitor on Linux Desktop:

**Snap**

[![Get it from the Snap Store](https://snapcraft.io/static/images/badges/en/snap-store-black.svg)](https://snapcraft.io/htop-pro)

Install with command-line

`snap install htop-pro`

and run

`htop-pro`
